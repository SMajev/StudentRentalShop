# StudentRentalShop


Console application in c# simulatin a univeristy equipment rental system

## Features
- Add users
- Add Equipment 
- Rent and return equipment
- Late return penalty
- User loans limits
- Reports And History


## Structure
- equipment
- users
- rental
- reports


## Desing decisions 
- Separation of logic into services
- Low coupling between classes
- Use of inheritance for domain objects

## Run
dotnet run
